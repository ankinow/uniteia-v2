# Cursor AI

Cursor is an AI-powered code editor built on VS Code that provides intelligent code completion, chat assistance, and codebase-aware suggestions.

## Key Features

The editor integrates AI directly into the coding workflow, allowing developers to:

- Ask questions about their codebase in natural language
- Generate code that understands the full project context
- Refactor across multiple files simultaneously
- Review AI-generated diffs inline before accepting changes

## Getting Started

1. Download Cursor from the official website
2. Import your existing VS Code settings
3. Start coding with AI assistance using `Cmd+K`
