# OpenCode: Build Software Without Knowing How to Code

You don't need to be a programmer to create software anymore. OpenCode is like having a senior developer inside your terminal — you describe what you want in plain English, and it builds it for you.

## What Is OpenCode?

Think of OpenCode as a smart assistant that lives in your computer's terminal. You type what you need in normal language, and it figures out the code, runs it, tests it, and fixes problems automatically.

No syntax to memorize. No programming languages to learn. Just tell it what you want.

### Why Vibecoders Love It

**Talk to it like a person.** "Make me a website for my bakery with a menu, photos, and a contact form" — and OpenCode builds it. You don't write code, you describe outcomes.

**It explains what it's doing.** OpenCode shows you every step. You learn by watching. Many vibecoders start knowing nothing and understand programming within weeks just by observing.

**It fixes its own mistakes.** If something breaks, OpenCode reads the error, figures out what went wrong, and fixes it. You don't need to debug.

**It remembers everything.** OpenCode keeps track of your project across sessions. Come back tomorrow and it knows exactly where you left off.

## What Can You Build?

- **Websites and landing pages** — portfolios, business sites, blogs
- **Automations** — collect emails, send notifications, sync data
- **Small apps** — to-do lists, calculators, inventory trackers
- **Bots and agents** — auto-responders, data collectors, monitors
- **Anything you can describe** — if you can explain it, OpenCode can build it

## How It Works (In Plain Terms)

```
You type: "Create a website for my cat's birthday party with RSVP"
           ↓
OpenCode:  Plans the structure, picks the tools, writes the code
           ↓
OpenCode:  Shows you what it built, asks if you like it
           ↓
You type:  "Make the background pink and add confetti animation"
           ↓
OpenCode:  Updates everything, shows you the result
           ↓
You type:  "Put it online so I can share the link"
           ↓
OpenCode:  Deploys it, gives you a URL to share
```

## Getting Started (No Experience Needed)

```bash
# Step 1: Install OpenCode (copy-paste this one line)
npm install -g opencode

# Step 2: Start your first project (describe anything you want)
opencode "I want to build a simple website for my photography portfolio"
```

That's it. Two commands. No setup. No configuration. No coding knowledge required.

## The Vibecoder Mindset

Traditional programmers write code line by line. Vibecoders describe outcomes and iterate. You're not "learning to code" — you're learning to direct an AI that codes for you. Think of it like directing a movie: you don't need to know how to operate the camera or edit film. You just need vision.

OpenCode is your production crew.
