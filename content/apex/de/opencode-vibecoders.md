# OpenCode: Erstelle Software Ohne Programmieren zu Können

Du musst kein Programmierer sein. OpenCode ist dein persönlicher Entwickler im Terminal.

OpenCode ist ein intelligenter Assistent in deinem Terminal. Beschreibe was du willst und er baut es. Kein Code, keine Syntax, keine Vorkenntnisse.

### Loslegen
```bash
npm install -g opencode
opencode "Ich möchte eine Website für mein Geschäft erstellen"
```

**Jetzt starten:** [opencode.ai/go](https://opencode.ai/go?ref=094WWK44VS)
