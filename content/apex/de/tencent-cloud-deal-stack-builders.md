# Tencent Cloud Deal Stack: Günstige Cloud für Builder

## Was ist Tencent Cloud Deal Stack?

Tencent Cloud bietet eine Reihe von Produkten und Promotionen, die ideal für unabhängige Builder sind. Mit Lighthouse, CVM und EdgeOne können Sie Websites, APIs, Bots, Dashboards und kleine Anwendungen hosten und dabei sehr wenig ausgeben — in einigen Fällen gar nichts.

## Kostenlose Produkte

Tencent Cloud bietet eine kostenlose Stufe mit ausgewählten Produkten. Kostenlose Ressourcen umfassen:

- **EdgeOne:** Bis zu 1M Anfragen/Monat im kostenlosen Plan
- **Lighthouse:** Testzeitraum bei Basiskonfigurationen
- **CVM:** Aktionsangebote für neue Benutzer

> ⚠️ **Hinweis:** Kostenlose Produkte erfordern möglicherweise die Registrierung einer Kreditkarte. Überprüfen Sie die offiziellen Bedingungen.

## Lighthouse — Einfachheit, die Funktioniert

Lighthouse ist ein vereinfachter VPS, ideal für diejenigen, die keine komplexe Infrastruktur verwalten möchten.

**Wann verwenden:**
- Statische Websites oder Blogs
- Leichtgewichtige APIs
- Bots und Automatisierung
- Entwicklungsumgebungen
- Persönliche Dashboards

**Vorteile:**
- Feste Spezifikationen, keine Überraschungen
- Vereinfachtes Dashboard
- Firewall und Monitoring inklusive
- Günstiger als vergleichbare CVM
- Monatliche oder stündliche Abrechnung

## CVM — Volle Leistung

CVM (Cloud Virtual Machine) ist die Komplettlösung für diejenigen, die volle Kontrolle benötigen.

**Wann verwenden:**
- CPU/RAM-intensive Anwendungen
- Kubernetes oder erweitertes Docker
- Benutzerdefinierter Kernel oder Netzwerk-Tuning
- Große Datenbanken
- Umgebungen, die VPC und Sicherheitsgruppen erfordern

**Vorteile:**
- Vollständig anpassbare Konfiguration
- Dedizierte, Spot- und reservierte Instanzen
- Zusätzlicher Blockspeicher
- Sekundengenaue Abrechnung (Minimum 1 Stunde)
- BYOL-Unterstützung

## EdgeOne — CDN + Sicherheit

EdgeOne kombiniert CDN mit WAF, DDoS-Schutz und Bot-Management in einer Plattform.

**Wann verwenden:**
- Globale Inhaltsbereitstellung beschleunigen
- Websites vor Angriffen schützen
- Separate CDN + WAF ersetzen
- Latenz für internationale Benutzer reduzieren

**Vorteile:**
- Großzügige kostenlose Stufe (1M Anfr./Monat)
- Pay-as-you-go
- Globales Edge-Netzwerk
- Native Integration mit Lighthouse und CVM
- Keine komplexen Lizenzen

## Wie man Produkte Kombiniert

| Stack | Lighthouse + EdgeOne | CVM + EdgeOne |
|-------|---------------------|---------------|
| Am besten für | Websites, Blogs, Landing Pages | Dynamische Apps, APIs, E-Commerce |
| Leistung | Großartig für statische Inhalte | Maximale Flexibilität |
| Kosten | Niedrigste | Moderat |
| Einrichtung | Minuten | Stunden |

## Bevor Sie Zahlen — Checkliste

1. **Region prüfen:** Nicht alle Promos sind in allen Regionen verfügbar
2. **Berechtigung:** Einige Angebote gelten nur für neue Benutzer
3. **Gültigkeit:** Promotionen laufen ab — Datum prüfen
4. **Verlängerung:** Promo-Preis gilt möglicherweise nicht bei Verlängerung
5. **Gutscheine:** Bedingungen vor Aktivierung lesen — einige erfordern Mindestausgaben
6. **Kostenlose Stufe:** Bestätigen, ob Kreditkarte erforderlich ist

## Empfohlene Einrichtung für Builder

### Website/Blog
Lighthouse (Basis) + EdgeOne (kostenlose Stufe)

### Leichtgewichtige API
Lighthouse (mittlerer Plan) + EdgeOne (kostenlose Stufe)

### Bot / Discord Bot
Lighthouse (Basis) + EdgeOne (kostenlose Stufe)

### Dashboard / Analytics
Lighthouse (mittlerer Plan) + EdgeOne (Pay-as-you-go)

### Vollständige Anwendung
CVM (Spot-Instanz) + EdgeOne (Pay-as-you-go)
