# OpenCode: Crea Software Sin Saber Programar

No necesitas ser programador. OpenCode es como tener un desarrollador en tu terminal.

OpenCode es un asistente inteligente en tu terminal. Describe lo que quieres en español y él lo construye. Sin código, sin sintaxis, sin experiencia previa.

### ¿Qué puedes crear?
Sitios web, landing pages, automatizaciones, mini apps, bots. Si puedes describirlo, OpenCode puede construirlo.

### ¿Cómo funciona?
Hablas con OpenCode como con una persona. Él planea, escribe el código, lo prueba y corrige errores solo. Tú solo describes y revisas.

### Comenzar (cero experiencia)
```bash
npm install -g opencode
opencode "Quiero crear un sitio web para mi negocio"
```
