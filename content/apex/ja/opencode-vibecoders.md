# OpenCode: コードの書き方を知らなくてもソフトウェアを構築

もうプログラマーでなくてもソフトウェアを作れます。OpenCodeは、ターミナルの中にシニア開発者がいるようなものです——普通の英語で欲しいものを説明すれば、それを構築してくれます。

## OpenCodeとは？

OpenCodeは、コンピュータのターミナルに住むスマートなアシスタントと考えてください。通常の言葉で必要なものを入力すれば、コードを考え、実行し、テストし、問題を自動的に修正します。

覚えるべき構文はありません。学ぶべきプログラミング言語もありません。ただ、欲しいものを伝えるだけです。

### バイブコーダーが愛する理由

**人と話すように使える。** 「パン屋のウェブサイトを作って。メニュー、写真、お問い合わせフォーム付き」——そう言えば、OpenCodeが構築してくれます。コードを書くのではなく、成果を説明するだけです。

**何をしているか説明してくれる。** OpenCodeはすべてのステップを表示します。見て学べます。多くのバイブコーダーは何も知らない状態から始め、観察するだけで数週間でプログラミングを理解できるようになります。

**自分のミスを修正してくれる。** 何か壊れたら、OpenCodeはエラーを読み、何が悪かったかを考え、修正します。デバッグする必要はありません。

**すべてを覚えてくれる。** OpenCodeはセッションを超えてプロジェクトを追跡します。明日戻ってきても、どこまで進めたか正確に把握しています。

## 何が構築できる？

- **ウェブサイトとランディングページ** — ポートフォリオ、ビジネスサイト、ブログ
- **自動化** — メール収集、通知送信、データ同期
- **小さなアプリ** — ToDoリスト、計算機、在庫管理ツール
- **ボットとエージェント** — 自動応答、データ収集、監視システム
- **説明できるものなら何でも** — 説明できれば、OpenCodeが構築できます

## 仕組み（簡単に説明）

```
あなたの入力: "猫の誕生日パーティー用のウェブサイトを作って。RSVP機能も欲しい"
           ↓
OpenCode:  構造を計画し、ツールを選び、コードを書く
           ↓
OpenCode:  作ったものを見せて、気に入ったかどうかを確認する
           ↓
あなたの入力:  "背景をピンクにして、紙吹雪のアニメーションを追加して"
           ↓
OpenCode:  すべてを更新し、結果を見せる
           ↓
あなたの入力:  "オンラインに公開して、リンクを共有できるようにして"
           ↓
OpenCode:  デプロイし、共有用のURLを提供する
```

## はじめ方（経験不要）

```bash
# ステップ1: OpenCodeをインストール（この1行をコピー＆ペーストするだけ）
npm install -g opencode

# ステップ2: 最初のプロジェクトを開始（何でも好きなことを書いてください）
opencode "写真ポートフォリオ用のシンプルなウェブサイトを作りたい"
```

以上です。2つのコマンドだけです。セットアップ不要。設定不要。コードの知識も不要。

## バイブコーダーの考え方

従来のプログラマーは1行ずつコードを書きます。バイブコーダーは成果を言語化し、反復改善します。あなたは「コードを学ぶ」のではなく、コードを書くAIを指導する方法を学んでいるのです。映画の監督に例えるとわかりやすいでしょう。カメラの操作法やフィルムの編集方法を知る必要はありません。ビジョンがあればいいのです。

OpenCodeがあなたの制作チームです。
