# OpenCode: プログラミングなしでソフトウェアを作ろう

プログラマーである必要はありません。OpenCodeはあなたのターミナルの中の開発者です。

OpenCodeはターミナルの中のスマートアシスタントです。日本語で欲しいものを説明すれば、自動で構築します。コードも文法も経験も不要です。

### はじめ方
```bash
npm install -g opencode
opencode "自分のビジネスのウェブサイトを作りたい"
```
