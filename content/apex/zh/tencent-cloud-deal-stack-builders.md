# Tencent Cloud Deal Stack：面向开发者的低价云服务

## 什么是Tencent Cloud Deal Stack？

腾讯云提供一系列适合独立开发者的产品和促销活动。通过Lighthouse、CVM和EdgeOne，您可以以极低的成本托管网站、API、机器人、仪表盘和小型应用程序——在某些情况下甚至可以完全免费。

## 免费产品

腾讯云提供精选产品的免费套餐。免费资源包括：

- **EdgeOne：** 免费计划每月最多100万次请求
- **Lighthouse：** 基础配置的试用期
- **CVM：** 新用户促销优惠

> ⚠️ **注意：** 免费产品可能需要注册信用卡。请查看官方条款。

## Lighthouse — 简便可用的云服务器

Lighthouse 是一款轻量级云服务器，非常适合不想管理复杂基础设施的用户。

**适用场景：**
- 静态网站或博客
- 轻量级API
- 机器人和自动化
- 开发环境
- 个人仪表盘

**优势：**
- 配置固定，无意外费用
- 简化的管理面板
- 内置防火墙和监控
- 比同等配置的CVM更便宜
- 按月或按小时计费

## CVM — 完整性能

CVM（云虚拟机）是为需要完全控制的用户提供的完整解决方案。

**适用场景：**
- CPU/内存密集型应用
- Kubernetes或高级Docker
- 自定义内核或网络调优
- 大型数据库
- 需要VPC和安全组的環境

**优势：**
- 完全可自定义的配置
- 专用实例、竞价实例和预留实例
- 额外块存储
- 按秒计费（最低1小时）
- 支持BYOL

## EdgeOne — CDN + 安全防护

EdgeOne将CDN、WAF、DDoS防护和机器人管理整合到一个平台中。

**适用场景：**
- 加速全球内容分发
- 保护网站免受攻击
- 替代独立的CDN + WAF方案
- 降低国际用户的延迟

**优势：**
- 慷慨的免费套餐（每月100万次请求）
- 按需付费
- 全球边缘网络
- 与Lighthouse和CVM原生集成
- 无需复杂授权

## 如何组合产品

| 组合 | Lighthouse + EdgeOne | CVM + EdgeOne |
|------|---------------------|---------------|
| 最佳用途 | 网站、博客、落地页 | 动态应用、API、电商 |
| 性能 | 静态内容表现出色 | 最大灵活性 |
| 成本 | 最低 | 适中 |
| 部署时间 | 数分钟 | 数小时 |

## 付费前检查清单

1. **确认区域：** 并非所有促销活动在所有区域都可用
2. **资格条件：** 部分优惠仅限新用户
3. **有效期：** 促销活动有时效——请检查日期
4. **续费：** 促销价格可能不适用于续费
5. **优惠券：** 激活前请阅读条款——部分要求最低消费
6. **免费套餐：** 确认是否需要信用卡
