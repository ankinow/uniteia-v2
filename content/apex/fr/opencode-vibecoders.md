# OpenCode: Créez des Logiciels Sans Savoir Coder

Pas besoin d'être programmeur. OpenCode est votre développeur personnel dans le terminal.

OpenCode est un assistant intelligent dans votre terminal. Décrivez ce que vous voulez en français et il le construit. Sans code, sans syntaxe, sans expérience.

### Commencer
```bash
npm install -g opencode
opencode "Je veux créer un site web pour mon entreprise"
```
