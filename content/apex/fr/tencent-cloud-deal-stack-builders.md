# Tencent Cloud Deal Stack : Cloud Pas Cher pour les Builders

## Qu'est-ce que Tencent Cloud Deal Stack ?

Tencent Cloud propose une gamme de produits et promotions idéaux pour les builders indépendants. Avec Lighthouse, CVM et EdgeOne, vous pouvez héberger des sites, APIs, bots, dashboards et petites applications en dépensant très peu — dans certains cas, rien du tout.

## Produits Gratuits

Tencent Cloud offre un niveau gratuit avec des produits sélectionnés. Les ressources gratuites incluent :

- **EdgeOne :** Jusqu'à 1M de requêtes/mois sur le plan gratuit
- **Lighthouse :** Période d'essai sur les configurations de base
- **CVM :** Offres promotionnelles pour les nouveaux utilisateurs

> ⚠️ **Note :** Les produits gratuits peuvent nécessiter l'enregistrement d'une carte de crédit. Vérifiez les termes officiels.

## Lighthouse — Simplicité qui Fonctionne

Lighthouse est un VPS simplifié, idéal pour ceux qui ne veulent pas gérer une infrastructure complexe.

**Quand utiliser :**
- Sites statiques ou blogs
- APIs légères
- Bots et automatisation
- Environnements de développement
- Dashboards personnels

**Avantages :**
- Spécifications fixes, pas de surprises
- Dashboard simplifié
- Firewall et monitoring inclus
- Moins cher que CVM équivalent
- Facturation mensuelle ou horaire

## CVM — Puissance Complète

CVM (Cloud Virtual Machine) est la solution complète pour ceux qui ont besoin d'un contrôle total.

**Quand utiliser :**
- Applications intensives en CPU/RAM
- Kubernetes ou Docker avancé
- Kernel personnalisé ou réglage réseau
- Grandes bases de données
- Environnements nécessitant VPC et groupes de sécurité

**Avantages :**
- Configuration entièrement personnalisable
- Instances dédiées, spot et réservées
- Stockage en bloc supplémentaire
- Facturation à la seconde (minimum 1 heure)
- Support BYOL

## EdgeOne — CDN + Sécurité

EdgeOne combine CDN avec WAF, protection DDoS et gestion des bots dans une seule plateforme.

**Quand utiliser :**
- Accélérer la livraison de contenu global
- Protéger les sites contre les attaques
- Remplacer CDN + WAF séparés
- Réduire la latence pour les utilisateurs internationaux

**Avantages :**
- Niveau gratuit généreux (1M req/mois)
- Paiement à l'usage
- Réseau de bord mondial
- Intégration native avec Lighthouse et CVM
- Pas de licences complexes

## Comment Combiner les Produits

| Stack | Lighthouse + EdgeOne | CVM + EdgeOne |
|-------|---------------------|---------------|
| Meilleur pour | Sites, blogs, landing pages | Apps dynamiques, APIs, e-commerce |
| Performance | Excellent pour contenu statique | Flexibilité maximale |
| Coût | Plus bas | Modéré |
| Configuration | Minutes | Heures |

## Avant de Payer — Checklist

1. **Vérifiez la région :** Toutes les promos ne sont pas disponibles dans toutes les régions
2. **Éligibilité :** Certaines offres sont réservées aux nouveaux utilisateurs
3. **Validité :** Les promotions expirent — vérifiez la date
4. **Renouvellement :** Le prix promo peut ne pas s'appliquer au renouvellement
5. **Coupons :** Lisez les termes avant d'activer — certains nécessitent une dépense minimale
6. **Niveau gratuit :** Confirmez si une carte de crédit est requise

## Configuration Recommandée pour les Builders

### Site/Blog
Lighthouse (basique) + EdgeOne (niveau gratuit)

### API Légère
Lighthouse (plan moyen) + EdgeOne (niveau gratuit)

### Bot / Bot Discord
Lighthouse (basique) + EdgeOne (niveau gratuit)

### Dashboard / Analytics
Lighthouse (plan moyen) + EdgeOne (paiement à l'usage)

### Application Complète
CVM (instance spot) + EdgeOne (paiement à l'usage)
