# OpenCode: Crea Software Senza Saper Programmare

Non serve essere programmatori. OpenCode è il tuo sviluppatore personale nel terminale.

OpenCode è un assistente intelligente nel tuo terminale. Descrivi cosa vuoi in italiano e lui lo costruisce. Nessun codice, nessuna sintassi, nessuna esperienza.

### Iniziare
```bash
npm install -g opencode
opencode "Voglio creare un sito web per la mia attività"
```
