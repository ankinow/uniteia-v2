# Tencent Cloud Deal Stack: Cloud Economica per Builder

## Cos'è Tencent Cloud Deal Stack?

Tencent Cloud offre una gamma di prodotti e promozioni ideali per builder indipendenti. Con Lighthouse, CVM e EdgeOne, puoi ospitare siti, API, bot, dashboard e piccole applicazioni spendendo molto poco — in alcuni casi, nulla.

## Prodotti Gratuiti

Tencent Cloud offre un livello gratuito con prodotti selezionati. Le risorse gratuite includono:

- **EdgeOne:** Fino a 1M di richieste/mese sul piano gratuito
- **Lighthouse:** Periodo di prova sulle configurazioni di base
- **CVM:** Offerte promozionali per nuovi utenti

> ⚠️ **Nota:** I prodotti gratuiti potrebbero richiedere la registrazione di una carta di credito. Verifica i termini ufficiali.

## Lighthouse — Semplicità che Funziona

Lighthouse è un VPS semplificato, ideale per chi non vuole gestire infrastrutture complesse.

**Quando usare:**
- Siti statici o blog
- API leggere
- Bot e automazione
- Ambienti di sviluppo
- Dashboard personali

**Vantaggi:**
- Specifiche fisse, nessuna sorpresa
- Dashboard semplificato
- Firewall e monitoraggio inclusi
- Più economico di CVM equivalente
- Fatturazione mensile o oraria

## CVM — Potenza Completa

CVM (Cloud Virtual Machine) è la soluzione completa per chi ha bisogno di controllo totale.

**Quando usare:**
- Applicazioni intensive CPU/RAM
- Kubernetes o Docker avanzato
- Kernel personalizzato o tuning di rete
- Grandi database
- Ambienti che richiedono VPC e gruppi di sicurezza

**Vantaggi:**
- Configurazione completamente personalizzabile
- Istanze dedicate, spot e riservate
- Storage a blocchi aggiuntivo
- Fatturazione al secondo (minimo 1 ora)
- Supporto BYOL

## EdgeOne — CDN + Sicurezza

EdgeOne combina CDN con WAF, protezione DDoS e gestione bot in un'unica piattaforma.

**Quando usare:**
- Accelerare la distribuzione di contenuti globali
- Proteggere i siti dagli attacchi
- Sostituire CDN + WAF separati
- Ridurre la latenza per utenti internazionali

**Vantaggi:**
- Livello gratuito generoso (1M req/mese)
- Pagamento a consumo
- Rete edge globale
- Integrazione nativa con Lighthouse e CVM
- Nessuna licenza complessa

## Come Combinare i Prodotti

| Stack | Lighthouse + EdgeOne | CVM + EdgeOne |
|-------|---------------------|---------------|
| Migliore per | Siti, blog, landing page | App dinamiche, API, e-commerce |
| Performance | Ottima per contenuti statici | Massima flessibilità |
| Costo | Più basso | Moderato |
| Configurazione | Minuti | Ore |

## Prima di Pagare — Checklist

1. **Verifica la regione:** Non tutte le promo sono disponibili in tutte le regioni
2. **Idoneità:** Alcune offerte sono solo per nuovi utenti
3. **Validità:** Le promozioni scadono — verifica la data
4. **Rinnovo:** Il prezzo promo potrebbe non applicarsi al rinnovo
5. **Coupon:** Leggi i termini prima di attivare — alcuni richiedono spesa minima
6. **Livello gratuito:** Conferma se è richiesta carta di credito
