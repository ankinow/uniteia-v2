# Tencent Cloud Deal Stack: Cloud Economica per Creator

## Cos'è Tencent Cloud Deal Stack?

Tencent Cloud offre una gamma di prodotti e promozioni ideali per creator indipendenti. Con Lighthouse, CVM e EdgeOne puoi ospitare siti, API, bot, dashboard e piccole applicazioni spendendo pochissimo — in alcuni casi, nulla.

## Prodotti Gratuiti

Tencent Cloud mette a disposizione un livello gratuito con prodotti selezionati. Le risorse gratuite includono:

- **EdgeOne:** Fino a 1M richieste/mese nel piano gratuito
- **Lighthouse:** Periodo di prova su configurazioni base
- **CVM:** Offerte promozionali per nuovi utenti

> ⚠️ **Nota:** I prodotti gratuiti potrebbero richiedere la registrazione con carta di credito. Verifica i termini ufficiali.

## Lighthouse — Semplicità che Funziona

Lighthouse è un VPS semplificato, ideale per chi non vuole gestire infrastrutture complesse.

**Quando usarlo:**
- Siti statici o blog
- API leggere
- Bot e automazione
- Ambienti di sviluppo
- Dashboard personali

**Vantaggi:**
- Specifiche fisse, nessuna sorpresa
- Dashboard semplificata
- Firewall e monitoraggio inclusi
- Più economico di un CVM equivalente
- Fatturazione mensile o oraria

## CVM — Potenza Completa

CVM (Cloud Virtual Machine) è la soluzione completa per chi necessita di controllo totale.

**Quando usarlo:**
- Applicazioni ad alto consumo di CPU/RAM
- Kubernetes o Docker avanzato
- Ottimizzazioni personalizzate di kernel o rete
- Database di grandi dimensioni
- Ambienti che richiedono VPC e gruppi di sicurezza

**Vantaggi:**
- Configurazione completamente personalizzabile
- Istanze dedicate, spot e riservate
- Storage a blocchi aggiuntivo
- Fatturazione al secondo (minimo 1 ora)
- Supporto BYOL

## EdgeOne — CDN + Sicurezza

EdgeOne combina CDN con WAF, protezione DDoS e gestione dei bot in un'unica piattaforma.

**Quando usarlo:**
- Accelerare la distribuzione di contenuti globali
- Proteggere i siti dagli attacchi
- Sostituire CDN + WAF separati
- Ridurre la latenza per utenti internazionali

**Vantaggi:**
- Livello gratuito generoso (1M richieste/mese)
- Pagamento a consumo
- Rete edge globale
- Integrazione nativa con Lighthouse e CVM
- Nessuna licenza complessa

## Come Combinare i Prodotti

| Stack | Lighthouse + EdgeOne | CVM + EdgeOne |
|-------|---------------------|---------------|
| Ideale per | Siti, blog, landing page | App dinamiche, API, e-commerce |
| Prestazioni | Ottime per contenuti statici | Massima flessibilità |
| Costo | Più basso | Moderato |
| Configurazione | Minuti | Ore |

## Prima di Pagare — Lista di Controllo

1. **Controlla la regione:** Non tutte le promozioni sono disponibili in tutte le regioni
2. **Idoneità:** Alcune offerte sono solo per nuovi utenti
3. **Validità:** Le promozioni scadono — verifica la data
4. **Rinnovo:** Il prezzo promozionale potrebbe non applicarsi al rinnovo
5. **Coupon:** Leggi i termini prima di attivare — alcuni richiedono una spesa minima
6. **Livello gratuito:** Conferma se è richiesta una carta di credito
