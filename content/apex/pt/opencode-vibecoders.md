# OpenCode: Crie Software Sem Saber Programar

Você não precisa ser programador para criar software. OpenCode é como ter um desenvolvedor sênior dentro do seu terminal — você descreve o que quer em português, e ele constrói pra você.

## O Que É OpenCode?

Pense no OpenCode como um assistente inteligente que mora no terminal do seu computador. Você digita o que precisa em linguagem normal, e ele descobre o código, executa, testa e corrige problemas sozinho.

Sem sintaxe pra decorar. Sem linguagens de programação pra aprender. Só dizer o que você quer.

### Por Que Vibecoders Amam

**Fale com ele como se fosse uma pessoa.** "Cria um site pra minha padaria com cardápio, fotos e formulário de contato" — e o OpenCode constrói. Você não escreve código, você descreve resultados.

**Ele explica o que está fazendo.** O OpenCode mostra cada passo. Você aprende observando. Muitos vibecoders começam sem saber nada e em semanas já entendem de programação só de assistir.

**Ele conserta os próprios erros.** Se algo quebrar, o OpenCode lê o erro, entende o que deu errado e conserta. Você não precisa fazer debug.

**Ele lembra de tudo.** O OpenCode guarda o contexto do seu projeto entre sessões. Volta amanhã e ele sabe exatamente onde você parou.

## O Que Dá Pra Construir?

- **Sites e landing pages** — portfólios, sites de negócios, blogs
- **Automações** — coletar emails, enviar notificações, sincronizar dados
- **Mini aplicativos** — listas de tarefas, calculadoras, controle de estoque
- **Bots e agentes** — respondidores automáticos, coletores de dados, monitores
- **Qualquer coisa que você consiga descrever** — se você sabe explicar, o OpenCode sabe construir

## Como Funciona (Em Termos Simples)

```
Você digita: "Cria um site pro aniversário do meu gato com confirmação de presença"
              ↓
OpenCode:    Planeja a estrutura, escolhe as ferramentas, escreve o código
              ↓
OpenCode:    Mostra o que construiu, pergunta se você gostou
              ↓
Você digita: "Deixa o fundo rosa e coloca animação de confete"
              ↓
OpenCode:    Atualiza tudo, mostra o resultado
              ↓
Você digita: "Publica online pra eu compartilhar o link"
              ↓
OpenCode:    Faz deploy, te dá uma URL pra compartilhar
```

## Começando (Zero Experiência Necessária)

```bash
# Passo 1: Instalar o OpenCode (copia e cola essa linha)
npm install -g opencode

# Passo 2: Começar seu primeiro projeto (descreve o que quiser)
opencode "Quero criar um site simples pro meu portfólio de fotografia"
```

Só isso. Dois comandos. Zero configuração. Zero conhecimento técnico.

## A Mentalidade Vibecoder

Programadores tradicionais escrevem código linha por linha. Vibecoders descrevem resultados e iteram. Você não está "aprendendo a programar" — está aprendendo a dirigir uma IA que programa pra você. Pense como se estivesse dirigindo um filme: você não precisa saber operar a câmera ou editar. Você só precisa ter visão.

OpenCode é sua equipe de produção.
